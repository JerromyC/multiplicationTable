Time to practice Multiplication!

It might seem like a drag sometimes, but knowing and memorizing these common Multiplication problems will make all math easier in the long run.

  1. You get 3 attemts per problem before its marked "Wrong"
  2. If you get one wrong (3x) it will cycle to the next problem
  3. After you are finished, make sure to email this to a parent/teacher or download the results for review

We recommend one practice a day, ideally before more difficult math work begins to wake the brain up a bit and get it in the mood for some mathin.
